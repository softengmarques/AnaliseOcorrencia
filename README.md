# Análise de Ocorrências Aéreas

Projeto de análise de dados aeronáuticos utilizando Python.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Git/GitHub

## Funcionalidades
- Leitura de CSV
- Limpeza de dados
- Análise de fatalidades
- Geração de gráficos
- Relatórios automáticos

```bash
pip install -r requirements.txt
